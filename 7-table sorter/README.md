## 二、Table Sorter

Table sorter包括JavaScript和一点CSS，能够让原始的html table变得可以分别按照各栏数据值，对各行排序。

### 效果

1. 在表头任意一个栏目中点击一下，下面各行将按照此栏目值的升序排序

   a. 按照字符串比较来确定顺序

2. 再次点击该栏目，变更为降序排序

3. 点击其它栏目，则按其它栏目的值重新排序

4. 注意，排序时，栏目奇偶行的背景色保持奇数白色、偶数浅灰色
